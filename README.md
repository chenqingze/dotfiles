# dotfiles
**A set of emacs, vim, zsh, git, and other configuration files.**

## Installation
### 1. Install rcm (for mac) 
 
```
brew install rcm
```
Other version and rcm refer :https://github.com/thoughtbot/rcm

### 2. Clone this repo to your $HOME/.dotfiles directory

```
git clone git@github.com:chenqingze/dotfiles.git .dotfiles
```
***please edit .gitconfig file ,make sure your github information is correct !!!***

### 3. run rcm commandline

```
rcdn -v
```
